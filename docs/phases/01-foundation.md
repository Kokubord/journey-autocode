# Foundation — setup operacional

Foundation **não** é descoberta de produto. É deixar o projecto **tecnicamente pronto** para a Discovery.

## Objectivo

- Repositório git (local; remoto opcional)
- Journey instalada e a responder (Opening Handshake)
- Artefactos base: `HANDOVER.md`, `AGENTS.md`, instruções do agente, constituição *placeholder*, CI mínimo, ADR-0001 de adopção

## O que NÃO fazer aqui

- Fechar visão de produto completa  
- Decidir stack definitiva  
- Escrever todas as specs  

Se a ideia ainda está mole, volta ao [Warmup](00-warmup.md).
