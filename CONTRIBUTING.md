# Contributing to Journey AutoCode

- Código e mensagens de commit: **inglês**.
- Documentação de produto/adopção: **português** (ou bilingue quando fizer sentido).
- PRs pequenos e revistos; CI verde.
- Não commits secrets (`.pem`, `.env`).
- Ao contribuir, concordas com o licenciamento **Apache-2.0** deste repositório.
